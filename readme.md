<p>Приложение, в реальном времени показывает/обновляет стоимость/%роста крипто-валют из разных источников.</p>

## Установка

**NPM**

```
npm install 
```
**Composer**

```
composer install 
```

**database**

```
Создать базу данных и запустить миграции   - php artisan migrate  
```

## Compatibility
PHP 7+
